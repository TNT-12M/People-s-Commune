## 技术栈

- **后端**: Node.js + Express
- **数据库**: SQLite (sqlite3)
- **实时通信**: WebSocket (ws库)
- **前端**: 原生HTML5 + CSS3 + JavaScript
- **密码处理**: bcryptjs (虽然简化版可能不需要)
- **会话管理**: express-session

## 核心功能模块

### 1. 数据库设计 (database.js)

创建以下表结构：

**users表** - 用户信息
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- nickname: TEXT UNIQUE NOT NULL (用户昵称)
- avatar: TEXT (头像URL，可选)
- grade: TEXT (当前选择的年级)
- created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

**messages表** - 聊天消息
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- user_id: INTEGER (外键关联users)
- grade: TEXT NOT NULL (年级)
- subject: TEXT NOT NULL (学科)
- content: TEXT NOT NULL (消息内容)
- created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

**grades表** - 年级数据（预置）
- id: INTEGER PRIMARY KEY
- name: TEXT NOT NULL (年级名称，如"一年级")
- level: TEXT NOT NULL (阶段：小学/初中/高中)
- sort_order: INTEGER (排序)

**subjects表** - 学科数据（预置）
- id: INTEGER PRIMARY KEY
- name: TEXT NOT NULL (学科名称)
- grade_id: INTEGER (外键关联grades)

初始化时预置K12年级和学科数据：
- 小学1-6年级：语文、数学、英语
- 初中1-3年级：语文、数学、英语、物理、化学、生物、历史、地理、政治
- 高中1-3年级：语文、数学、英语、物理、化学、生物、历史、地理、政治

### 2. 后端服务器 (server.js)

**Express路由**:
- `GET /` - 返回主页面
- `POST /api/register` - 用户注册（检查昵称唯一性）
- `POST /api/login` - 用户登录（验证昵称存在）
- `GET /api/grades` - 获取所有年级列表
- `GET /api/subjects/:gradeId` - 获取指定年级的学科列表
- `GET /api/messages/:grade/:subject` - 获取指定年级学科的聊天记录
- `POST /api/messages` - 发送新消息

**WebSocket服务**:
- 监听连接事件，维护在线用户列表
- 接收客户端发送的消息，保存到数据库
- 广播消息给同一房间（年级+学科）的所有在线用户
- 处理用户断开连接

**中间件**:
- 静态文件服务（public目录）
- JSON body解析
- Session管理
- CORS支持

### 6. 实时通信流程

**WebSocket消息格式**:
```javascript
// 客户端发送
{
  type: 'message',
  data: {
    grade: '一年级',
    subject: '语文',
    content: '消息内容'
  }
}

// 服务端广播
{
  type: 'new_message',
  data: {
    id: 1,
    userId: 1,
    nickname: '用户昵称',
    grade: '一年级',
    subject: '语文',
    content: '消息内容',
    createdAt: '2026-04-06 10:00:00'
  }
}
```

**房间机制**:
- 每个"年级+学科"组合是一个独立的聊天房间
- 用户选择学科后自动加入对应房间
- 只接收当前房间的消息广播

## 关键代码示例

**WebSocket房间管理**:
```javascript
const rooms = new Map(); // roomKey -> Set of WebSocket connections

function joinRoom(ws, grade, subject) {
  const roomKey = `${grade}_${subject}`;
  if (!rooms.has(roomKey)) {
    rooms.set(roomKey, new Set());
  }
  rooms.get(roomKey).add(ws);
  ws.currentRoom = roomKey;
}

function broadcastToRoom(roomKey, message) {
  const room = rooms.get(roomKey);
  if (room) {
    room.forEach(client => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(JSON.stringify(message));
      }
    });
  }
}
```

**消息渲染**:
```javascript
function renderMessage(msg, isOwn) {
  const div = document.createElement('div');
  div.className = `message ${isOwn ? 'own' : 'other'}`;
  div.innerHTML = `
    <div class="message-header">
      <span class="nickname">${msg.nickname}</span>
      <span class="time">${formatTime(msg.createdAt)}</span>
    </div>
    <div class="message-content">${escapeHtml(msg.content)}</div>
  `;
  return div;
}
```
